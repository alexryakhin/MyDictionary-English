개인정보 처리방침
최종 수정: 2025년 11월

소개

My Dictionary에 오신 것을 환영합니다. 귀하의 개인정보를 존중하며 개인 데이터를 보호합니다. 본 방침은 단어장 앱 이용 시 정보를 어떻게 수집·이용·보호하는지 설명합니다.

요약: 앱 운영에 필요한 데이터만 수집하며, 개인정보를 판매하지 않으며, 귀하는 데이터를 완전히 관리할 수 있습니다.

수집하는 정보

개인 데이터

  • 이메일: 계정 생성 및 인증
  • 표시 이름: 선택 사항, 맞춤화용
  • 인증 데이터: Google/Apple을 통한 안전한 로그인

앱 이용 데이터

  • 단어장 단어: 단어장에 추가한 단어
  • 학습 진도: 테스트 결과 및 학습 패턴
  • 설정: 선호 및 맞춤화
  • 기기 정보: 기기 종류, OS·앱 버전

AI 콘텐츠 및 수업 데이터

  • 요청 및 입력: 정의·테스트·AI 스토리용으로 보내는 단어·구문·주제
  • AI 응답: OpenAI 모델이 생성한 결과(정의, 스토리, 수업, 추천)
  • 맞춤화 맥락: 결과 조정을 위한 선택적 프로필 데이터(목표, 관심사, 언어)

진단·분석 데이터

  • Firebase Authentication: 세션용 익명 ID
  • 크래시·성능 로그: 개인 콘텐츠 없이 기기 진단
  • 구독 상태: 접근 확인을 위한 RevenueCat ID 및 영수증

공유 단어장 데이터

  • 협업 콘텐츠: 공유 단어장 내 단어
  • 참가자 정보: 참가자 이메일 주소
  • 활동: 공유 단어장 내 기여 및 상호작용

정보 이용 방법

기본 기능

  • 단어장 학습 기능 제공
  • 기기 간 데이터 동기화
  • 공유 단어장 운영
  • 이용 경험 맞춤화

개선 및 분석

  • 이용 분석을 통한 기능 개선
  • 기술적 문제 탐지 및 해결
  • 새 도구 개발
  • 성능 최적화

소통

  • 업데이트 및 중요 공지 전송
  • 이용자 지원 제공
  • 협업 초대 전송

데이터 공유 및 제3자

개인정보를 제3자에게 판매·교환·양도하지 않습니다.

서비스 제공자

  • Firebase (Google): 인증, 원격 설정, 안전한 저장
  • RevenueCat: App Store·Google Play 구독 확인
  • OpenAI: 요청 처리 및 교육 콘텐츠 생성
  • Google / Apple Sign-In: 연동 인증

협업 기능

  • 이메일 주소는 단어장 참가자에게만 공유됩니다
  • 공유 콘텐츠는 모든 참가자가 볼 수 있습니다
  • 공유 단어장 접근 권한은 귀하가 관리합니다

제3자 서비스의 데이터 처리

신뢰할 수 있는 제공자를 사용합니다. 각 제공자는 필요한 최소 데이터만 받습니다.

OpenAI

  • 요청·선택 단어·맥락을 받아 교육용 응답 생성
  • 이메일 및 결제 데이터는 받지 않습니다
  • 응답은 앱 내 기록을 위해 계정에만 저장됩니다

Firebase

  • 안전한 인증, 분석, 원격 설정
  • 저장된 단어장이 아닌 암호화된 ID 및 기기 메타데이터 저장
  • 오용 탐지, 서비스 유지, 업데이트 전달에 사용

RevenueCat

  • 구독 관리를 위해 구매 영수증 및 익명 사용자 ID 수신
  • 단어장 또는 AI 콘텐츠는 처리하지 않습니다
  • iOS·macOS·Android에서 권한 동기화 지원

데이터 보안

보호 조치

  • 전송 시 암호화
  • 보안 기준에 따른 클라우드 저장
  • 정기적인 보안 점검 및 업데이트
  • 접근 제어 및 인증

귀하의 유의 사항

  • 로그인 정보를 안전하게 보관하세요
  • 단어장 접근 권한은 신중히 부여하세요
  • 공용 기기에서는 로그아웃하세요

귀하의 권리와 선택

데이터 접근

  • 언제든 데이터를 보고 다운로드할 수 있습니다
  • 단어장 및 진도 내보내기
  • 계정 정보 확인

데이터 관리

  • 계정 및 관련 데이터 삭제
  • 공유 단어장에서 나가기
  • 선택적 분석·데이터 수집 거부
  • 알림 설정 관리

계정 관리

  • 프로필 업데이트
  • 개인정보 설정 변경
  • 구독 관리

데이터 보존

활성 계정

계정이 활성이고 서비스를 이용하는 동안 데이터를 보존합니다.

계정 삭제

  • 데이터는 30일 이내에 삭제됩니다
  • 다른 참가자가 활성인 동안 공유 콘텐츠가 남을 수 있습니다
  • 백업은 90일 이내에 삭제됩니다

법적 의무

법률 준수, 분쟁 해결, 계약 이행을 위해 일부 정보를 보존할 수 있습니다.

아동 개인정보

My Dictionary는 13세 미만을 대상으로 하지 않습니다. 13세 미만 아동의 개인 데이터를 고의로 수집하지 않습니다. 13세 미만 아동의 정보를 수집한 것으로 보이면 즉시 연락해 주세요.

국제 데이터 전송

데이터는 다른 국가에서 처리될 수 있습니다. Firebase, OpenAI, RevenueCat은 미국 등 다른 지역의 서버를 사용할 수 있습니다. 계약상 보장, 전송 시 암호화, 표준 보호 조치로 정보 안전을 유지합니다.

AI 콘텐츠 관리

앱 개인정보 설정에서 언제든 AI 맞춤화를 끌 수 있습니다. 끄면 프로필 데이터는 OpenAI에 전송되지 않지만, AI 기능을 위해 익명 요청은 필요합니다.

본 방침의 변경

본 개인정보 처리방침을 업데이트할 수 있습니다. 중요한 변경은 다음 방법으로 안내합니다:

  • 앱 내 새 방침 공개
  • 이메일 안내
  • 앱 내 알림

변경 후에도 앱을 계속 이용하면 업데이트된 방침에 동의한 것으로 봅니다.
